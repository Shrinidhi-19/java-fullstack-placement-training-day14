# React Hooks: useState & useEffect

## Overview

This project was developed as part of Java Full Stack Development Placement Training to learn React Hooks and state management.

The project demonstrates:

* useState Hook
* useEffect Hook
* Dependency Array
* State Management
* Event Handling
* Counter Application
* Side Effects in React
* Timer-based Updates

The examples help in understanding how React components manage state and perform actions during different stages of their lifecycle.

## Concepts Covered

### useState Hook

The useState Hook is used to create and manage component state.

Features Practiced:

* State Initialization
* State Updates
* Dynamic Rendering
* Event-Based State Changes

A counter application was created using useState to increment, decrement, and reset values.

---

### Counter Application

Implemented a simple counter that demonstrates state updates.

Operations:

* Increment Count
* Decrement Count
* Reset Count

The counter value updates automatically whenever the state changes.

---

### useEffect Hook

The useEffect Hook is used to perform side effects in React components.

Features Practiced:

* Running code after component rendering
* Delayed execution using timers
* Cleanup functions
* Lifecycle-like behavior

A timer-based example updates the displayed message after 4 seconds using useEffect.

---

### Dependency Array

Learned how dependency arrays control when useEffect executes.

Example:

* Effect runs whenever the count value changes.
* Console logging and alerts are triggered based on state updates.

The dependency array ensures the effect executes only when specific values change.

---

### Side Effects

Practiced handling side effects such as:

* Alerts
* Console Messages
* Timers
* Dynamic Updates

## These actions are executed using the useEffect Hook whenever the component renders or dependencies change.

### Cleanup Function

Implemented cleanup functionality to prevent memory leaks.

Concept Learned:

* Returning a cleanup function from useEffect
* Clearing timers when components unmount

This ensures efficient resource management in React applications.
